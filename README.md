# Бексеит және Жансайдың Беташар Той Сайты

💍 Беташар той үшін құрылған қарапайым және әрекетті сайт.

## Ақпарат

- **Жүргіндер**: Бексеит және Жансай
- **Күні**: 5 Шілде, 2026 (11:00)
- **Орны**: Кемел 2/1

## Функциялар

✨ **Светлі және романтикалық дизайн**
- Ақ түстің фондарында қалмақты оттеңіздер
- Адамға ұнайтын интерфейс
- Мобильді-жарамды

📝 **RSVP Форма**
- Қонақтардың атын өңейлік құру
- Электронды пошта және телефон мәліметтерін жинау
- Қонақтар санын беліліу
- Келу-келмеу жағдайын орнату

## Қалыпты ету

### Шарттар
- Node.js 16+
- npm немесе yarn

### Орнату

```bash
# Тәуелділіктерді орнату
npm install

# Дамуда сервер іске қосу
npm run dev

# http://localhost:3000 ішіңіз
```

### Өндіру

```bash
# Өндіру үшін құрылу
npm run build

# Өндіру сервер іске қосу
npm start
```
